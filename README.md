# SIMS-PPOB App
---
## About this project
A project made for take-home assignment to meet recruitment requirements
* Language : Typescript
* Library : ReactJS
* Framework : NextJS
* Global State Management : Redux
* Validation : Zod
* Styling : Tailwind CSS
Deployed on Railway
